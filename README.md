# Netflix-EDA-Project
Exploratory Data Analysis of Netflix Movies and TV Shows Dataset
## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies & TV Shows dataset.

https://colab.research.google.com/drive/1-xTfmSpOG5zuqu06maEURjOEt3EHSTgy?usp=sharing

## Objectives

* Clean and preprocess the dataset
* Explore trends and patterns
* Create data visualizations
* Generate business insights

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Dataset

Netflix Movies & TV Shows Dataset from Kaggle.

## Key Findings

1. Movies are more common than TV Shows.
2. The United States contributes the most content.
3. Netflix content increased rapidly after 2015.
4. TV-MA is the most common content rating.
5. Content production has grown significantly over time.
